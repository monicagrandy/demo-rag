### System Architecture Overview

- Structure agent enables catalog search functionality
  - Similar to UberEats/DoorDash menu search capabilities
  - Semantic search across restaurant offerings (burgers, vegan options, etc.)
  - User authentication for order lookup via order number
- Orchestrator acts as router based on user intent
  - Routes to menu agent for catalog searches
  - Routes to order agent for status inquiries
  - Handles hybrid queries requiring both agents
  - Synthesizes combined outputs from multiple agents

### Data Structure & Tools

- Menu data includes 8 items with:
  - Product ID, name, category, cuisine, price, rating
  - Dietary tags, description, availability status
- Order database contains 5 orders (IDs 201-205)
  - Order details: item ID, customer email, status, price
  - Order/delivery dates and tracking IDs
- Delivery policies integrated via prompt:
  - Just placed: 100% refund (1-2 business days)
  - Preparing: 50% refund (2-3 business days)
  - Out for delivery/delivered: No refund
- Vector DB implementation:
  - Embeds dish category, price, dietary info, description
  - Metadata includes ID, name, price for filtering
  - Returns top 3 relevant items per query
- Order lookup tools support:
  - Primary: Order ID matching
  - Secondary: Tracking ID matching
  - Tertiary: Customer email matching

### Voice Integration & Deployment

- Speech-to-text: OpenAI Whisper (most affordable option)
  - Word error rates: 10-12% decent, 5-6% very good, 2-3% elite
  - Hardware dependency affects quality (MacBook Air vs M4 Pro)
- Text-to-speech: OpenAI TTS-1
  - 11Labs alternative offers higher quality but much more expensive
  - Voice cloning available (requires 5-6 minutes of sample audio)
- Technical implementation:
  - Sound device library for hardware microphone access
  - Sound file library for wave file conversion
  - Real-time deployment would use webhooks vs current API approach
- Deployment via LangServe:
  - Built on FastAPI for high-performance endpoints
  - Direct integration with LangChain/LangGraph applications
  - Open source solution for production deployment
